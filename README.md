# cognisyn-bash


## Installation

Install the project using 

```bash
  git clone https://github.com/Cognisyn/cognisyn-bash /bin/ 
```
    
## Get started 
adding alias into bash 

``` bash 
cd /bin/cognisyn-bash
chmod +x ./test.sh
./test.sh

```

if you get an error -permisson denied

``` bash
su root

```
